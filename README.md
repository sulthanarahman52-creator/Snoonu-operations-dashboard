# Snoonu Operations Dashboard

## Project Overview
This Power BI dashboard analyzes food delivery operations inspired by Snoonu platform.

The dashboard provides insights into:
- Revenue trends
- Peak order hours
- Delivery performance
- Vendor analysis
- Location-wise orders
- Order status analysis

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key KPIs
- Total Revenue
- Total Orders
- Average Delivery Time
- Total Discount
- Average Order Value

## Dashboard Features
- Interactive slicers
- Peak hour analysis
- Revenue trend visualization
- Delivery status tracking
- Vendor performance analysis

## Insights
- Peak orders occur between 7 PM and 9 PM
- Doha generates the highest order volume
- Delivered orders represent nearly 80% of total orders
